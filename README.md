# Chuggers Horse Racing

Deployable MVP for the Chuggers Racing League.

## Stack
- Next.js
- Vercel
- Supabase

## Pages included
- Dashboard
- Festivals
- Race cards
- Horse index
- Hall of fame
- Stables
- Admin
- Login

## 1. Run locally
```bash
npm install
npm run dev
```

## 2. Supabase setup
Create a Supabase project, then copy the project URL and anon key into `.env.local` based on `.env.example`.

## 3. GitHub
Push this project to your repo: `chuggers-horse-racing`

## 4. Vercel
Import the GitHub repo into Vercel and add the same environment variables there.

## 5. Current state
This is a proper deployable starter with seeded league data in code. Authentication and database wiring are prepared structurally, but still intentionally simple for the first deployment pass.
